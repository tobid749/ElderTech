instrucciones claras para ejecutar el proyecto localmente.
    1- Copiar el link del github (https://github.com/tobid749/ElderTech.git)
    2-Abrir Visual Studio Code
    3- Insertar el link para clonar el proyecto
    4- Clonar el Proyecto
    5- Tocar Ctrl + Shift + P
    6- Apretar la opción Terminal Select Default Profile
    7- Apretar Ctrl + Ñ
    8- Poner dotnet watch run 
    9- Se ejecuta el programa!
